# @dimaslanjaka/ai-toolkit
Collection of AI-powered tools, agents, automations, and developer utilities.

## Features

### OpenAI-Compatible Server
- Full OpenAI API compatibility for chat completions, responses, completions, and embeddings
- Multiple provider support: OpenCode, Puter, ChatGPT
- **VS Code GitHub Copilot Support** — Streaming responses with required OpenAI fields (id, object, created, model, index, finish_reason) and `stream_options` support for usage reporting
- Web-based chat interface with conversation persistence
- Proxy checker for HTTP/SOCKS proxy validation
- SQLite database for model and proxy management
